***配件內容 : 遊戲計分圖 x 1,計時沙漏 x 1,鉛筆 x 1,詞語卡牌 x 100（共800個詞語）


規則簡介

  1. 1. 洗牌後將詞語牌庫朝下放在遊戲計分圖旁。玩家組成最少兩個成員的隊伍(最少兩隊)，商量好回合的先後順序。每隊取1張遊戲計分圖，並在第一個劃掉。
  2. 2. 輪到你那一隊時，選擇一位玩家擔任「描述者」（每個人都要輪到），手持一疊詞語卡，不可給其他人看到答案。
  3. 3. 翻轉沙漏，你們那隊的回合正式開始！
  4. 4. 描述者必須按己隊前回合在計分圖最後劃掉的格子所顯示之數字，描述牌上對應詞語。，怎麼講都行，但就是不可以講到答案上的文字！
  5. 5. 時間到即結算該回合的成果；按照答對結果，向前劃出等量格數，若出現負數現況，則視作沒有得分。
  6. 6. 隊伍描述者必須輪流替換，讓每名隊友都有機會擔當。
  7. 7. 最先到達終點(劃掉權全部空格)的一隊獲勝。
